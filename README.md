# 알고리즘 스터디 규칙
- 사용언어 : free
- 스터디 이전 : **알고리즘 문제 풀어오기**
- 스터디 당일 : **문제풀이 설명&발표** 및 **1~2문제 즉석에서 풀기**
- 문제 선정 : 각 주차에 맞게 개인당 2문제씩 선정한다.
- 풀이 올리기 : 각 개인 repository에 문제 풀이 코드를 올린다.
